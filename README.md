# xiaohongshu-content-studio

> Native-feeling Xiaohongshu content packaging for titles, body copy, covers, and carousels.

A Codex skill for building Xiaohongshu-ready content packages including titles, cover text, body copy, carousel direction, image ideas, and posting angles.

## What This Skill Does

This skill helps create Xiaohongshu content that feels native to the platform rather than generic AI writing. It is designed for project sharing, personal branding, study notes, AI tool posts, experience sharing, and other content that needs a real-person voice.

## Best For

- Xiaohongshu note ideas
- title and hook generation
- cover text
- body copy drafting
- carousel or image direction
- project packaging for public sharing

## Inputs

Typical inputs:
- topic, project, tool, or experience to share
- persona and target audience
- content goal such as traffic, trust, education, or discussion
- any screenshots, examples, or data points
- the specific deliverable needed

## Outputs

Typical outputs:
- multiple title options
- cover text options
- post angle and outline
- polished body copy
- image or carousel directions
- optional comment CTA suggestions

## Non-goals

This skill is not aimed at:
- formal documentation
- generic SEO blog writing
- image generation without content context
- sterile corporate voice

## Example Prompts

- `把我的校园招聘推荐系统包装成一篇小红书笔记。`
- `帮我写一篇本地部署 Qwen 的真实使用分享。`
- `给我 10 个封面标题和 3 套轮播图方向。`
- `把这个项目做成更像小红书原生内容的表达。`

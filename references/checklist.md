# Xiaohongshu Post Package Checklist

## Core Package

- 3 to 5 title options
- 1 to 3 cover text options
- clear post angle
- structured body copy
- visual or carousel direction

## Writing Standard

- Feels like a person, not a brochure
- Specific enough to trust
- Short enough to scan
- Strong opening without fake urgency

## Visual Standard

- Cover text is readable at small size
- One dominant idea per image
- Color and type direction match the persona

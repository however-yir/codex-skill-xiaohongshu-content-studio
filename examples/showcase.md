# Showcase Examples

These examples are designed to feel publishable and platform-native, not like generic AI-written social copy.

## Example 1: Project Sharing Post

**Prompt**

`把我的校园招聘推荐系统包装成一篇像真人写的小红书项目分享，带封面文案和轮播图方向。`

**A strong output should include**

- a believable personal angle
- native-feeling title and cover text
- body copy that turns system work into readable scenes
- carousel suggestions matched to the content rhythm

**Why this showcases the skill**

It proves the skill can translate technical work into social content without flattening it into cliché.

## Example 2: Real-use AI Tools Note

**Prompt**

`写一篇我本地部署 Qwen 的真实体验，重点是轻量使用、踩坑和我的判断，不要太像营销文。`

**A strong output should include**

- grounded, first-person phrasing
- specific scenes, tradeoffs, or constraints
- hooks that feel lived-in rather than exaggerated
- tone control that reduces obvious AI writing patterns

**Why this showcases the skill**

This example shows the skill's strength in authenticity and platform-native tone.

## Example 3: Before-and-after Workflow Content

**Prompt**

`帮我做一篇“用了 Codex 工作流前后变化”的小红书轮播内容，给我标题、封面、正文和每页结构。`

**A strong output should include**

- strong before-and-after framing
- crisp carousel pacing
- concrete improvement points instead of empty praise
- a call to action that fits Xiaohongshu naturally

**Why this showcases the skill**

It demonstrates that the skill can package workflows and personal productivity changes into engaging content structure.
